Automation

Below you will find some resources that deal with the subject of automation in Prism, so as to make it easier to reuse existing pieces of code from the guidance provided by Prism.

  • Composite Application Guidance for WPF & SL (Prism-v2) C# Snippets with Guidance, Links and Tips
    • Version: 2.0
    • Description: This blog post provides a code snippets with guidance, links & tips to perform common tasks in Prism.
  • Prism Automation: Templates for implementing the MVVM pattern using the Composite Application Library
    • Version: 2.0
    • Description: This blog post provides a template to create Views and ViewModels with interfaces.
  • C# Code Snippet to Create Commands with Attached Behaviors using Prism
    • Version: 2.0
    • Description: This blog post provides an installer for a C# snippet that enables easy creation of commands with attached behaviors.
  • Prism Quick Start Kit Update
    • Version: 2.0
    • Description: David Hill updated the Prism Quick Start Kit and added a WPF quick-start solution and WPF project templates.
  • Prism Quick Start Kit
    • Version: 2.0
    • Description: David Hill, p&p architect, created templates for Silverlight to create new solutions, modules and Shell projects using Prism-v2. This is an experiment of the p&p team on how to provide some level of automation and they are looking for your feedback.
  • Using the VS Templates for Composite-WPF-Guidance
    • Version: 1.0
    • Description: Visual Studio Templates for the Composite Application Guidance for WPF covering the New Solution, New Module and New View cases.
  • Composite Application Guidance for WPF Visual Studio Templates
    • Version: 1.0
    • Description: Visual Studio Templates to help developers to start with the creation of a solution using the Composite Application Guidance for WPF.
