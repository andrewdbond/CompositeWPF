Samples and Extensions

Below you will find some samples and extensions that demonstrate the use of prism and provide additional tools to extend its functionality.

  • Prism v2.1: Creating a Region Adapter for the Accordion control
    • Version: 2.0 and higher
    • Description: This blog post provides a sample to use SL4 Accordion control as a region.
  • Creating a multi-shell application in Prism-v2
    • Version: 2.0
    • Description: This blog post provides a sample WPF applications with multiple Shells.
  • Supporting Graceful Shutdown and Saving on Close From a WPF Prism App
    • Version: 2.0
    • Description: This blog post explains how to handle global operations in a Prism WPF application.
  • Prism And .Net RIA Services
    • Version: 2.0
    • Description: This blog post provides an approach about how to use RIA services in a Prism application.
  • C# Code Snippet to Create Commands with Attached Behaviors using Prism
    • Version: 2.0
    • Description: This blog post provides an installer for a C# snippet that enables easy creation of commands with attached behaviors.
  • Composite Application Guidance (Prism-v2) sample application using Silverlight 3 Child Window
    • Version: 2.0
    • Description: This blog post provides a sample to use the Silverlight 3 Child Window in a Prism-v2 application.
  • How to: Drag and Drop Silverlight Views between Regions in Prism-v2
    • Version: 2.0
    • Description: This blog post explains how to drag and drop Silverlight views between regions and provides a sample.
  • Sample Prism-v2 WPF Application showing WebBrowser control interaction
    • Version: 2.0
    • Description: Demo application that shows how to get a WebBrowser control and the sites it hosts to interact with a desktop application.
  • How to: Drag and Drop WPF Views between Regions in Prism-v2
    • Version: 2.0
    • Description: This blog post explains how to drag and drop WPF views between regions and provides a sample.
  • Prism-v2 Reference Implementation migrated to Silverlight 3 with Out-Of-Browser Capabilities
    • Version: 2.0
    • Description: This blog post details how to migrate the Prism-v2 RI to Silverlight 3 and add out-of-browser capabilities to it.
  • Composite Extensions for Prism 2
    • Version: 2.0
    • Description: These extensions allow you to use the modular loading patterns and capabilities of Prism as well as the pub/sub events in a Windows Forms application, or any other kind of application (even console apps, WCF services, etc.).
  • How to Add a View to a Region in a particular index with Prism-v2
    • Version: 2.0
    • Description: Article that explains how to Add a View to a Region in a particular index with Prism-v2 using an extension method for the region manager.
  • How To: Integrate a Prism v2 application with the Silverlight 3 Navigation Framework
    • Version: 2.0
    • Description: This blog post details how to use Silverlight 3 Navigation Framework and Prism-v2.
  • Composite Application Guidance for WPF & Silverlight (Prism v2) Reference Implementation migrated to XBap
    • Version: 2.0
    • Description: This post explains how to migrate the Prism-v2 RI to XBAP.
  • Using CAL with XBAP Applications
    • Version: 2.0
    • Description: This post shows the required steps to use the Composite Application Library with XBAP Applications.
  • Composite Extensions / Test Suite reference implementation
    • Version: 1.0
    • Description: This project provides a new sample application for the Composite WPF libraries. Also includes "Composite Extensions library" created during the development process of this application.
  • How to build an outlook style application
    • Version: 2.0
    • Description: This post by Erwin Van der Valk explains how to create an Outlook style application and introduces a couple of approaches for this.
  • How to build an outlook style application - Part 2
    • Version: 2.0
    • Description: This is the second part of Erwin Van der Valk's How to build an outlook style application where he includes popup windows and explains the approaches for the Outlook style application.
  • Composite WPF Contrib project
    • Version: Any
    • Description: Composite WPF Contrib is a community-developed library of extensions to the patterns & practices Composite WPF & SL project.
  • Blendability for Prism
    • Version: 4.0
    • Description: A blog post explaining a possible approach to allow design-time support for Prism.
  • Sofa: Prism + AvalonDock
    • Version: 4.0
    • Description: A project providing a wrapper of AvalonDock that integrates with Prism's architecture.
